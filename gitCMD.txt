Git terminal CMD :

pwd : affiche le chemin actuel 

ls : affiche les éléments présent dans le répertoire 

cd : pour changer de répertoire 

cd .. : pour revenir dans le répertoire père 

git status : affiche l'état du dossier clone //à jour ou pas 

git pull : sync

Pour push via command line :
git add . : pour cibler tout les fichiers à MAJ

git commit -m "Message descrp du push"

git push
